## 🌟 Overview

An **AI-powered Spam Detection Web App** built using **Flask + Machine Learning** that classifies messages as:

✔ SPAM 🚨  
✔ NORMAL ✅  

It uses **TF-IDF + Logistic Regression** for text classification and provides a modern dashboard with live statistics.

---

## 🎯 Features

✨ Upload CSV file for bulk prediction  
✨ AI-based spam detection  
✨ Real-time dashboard  
✨ Live Chart.js visualization 📊  
✨ SQLite database storage 🗄️  
✨ User Login & Registration 🔐  
✨ Clean modern UI 🌐
